# 📋 Module 7: Core JavaScript

## 🛠️ Phase 1: Environment Setup & Orientation

- [ ] **Task Explanation Video:** Review the official overview video to understand specific module delivery guidelines.
- [ ] **IDE Exploration & Selection:** Research, compare, and experiment with various Integrated Development Environments (e.g., VS Code, WebStorm).
  - [ ] Compare features: code editing, debugging tools, version control (Git) integration, language support, and UI customizable extensions.
  - [ ] Select and configure the primary workspace environment.

## 🧱 Phase 2: JavaScript Syntax & Flow Control

- [ ] **Variables & Memory Allocation:** Master the scope, mutability, and usage of `var`, `let`, and `const`.
- [ ] **Data Types & Type Conversions:** Implement primitive and non-primitive types, and manage implicit/explicit type conversion mechanics.
- [ ] **Operators:** Apply arithmetic, assignment, logical, comparison (`==` vs `===`), and bitwise operations.
- [ ] **Conditional Statements:** Build logical branching using `if`, `else if`, `else`, `switch` blocks, and clean conditional assignments via the ternary operator.
- [ ] **Looping Statements:** Implement standard control flow iterations using `for`, `while`, and `do...while` loops.

## 🧩 Phase 3: Data Structures & Advanced Iteration

- [ ] **Functions:** Write standard declarations, expressions, arrow functions, and parameter passing.
- [ ] **Objects:** Model real-world data structures using key-value pairs, access properties, and understand reference-type behavior.
- [ ] **Arrays:** Initialize, index, and manipulate structured collections of data.
- [ ] **Array Iteration Methods:** Build functional programming habits by mastering:
  - [ ] `forEach`
  - [ ] `map`
  - [ ] `filter`
  - [ ] `reduce`
  - [ ] `find`
- [ ] **String Methods:** Leverage native methods for text processing (e.g., slicing, concatenation, searching, case adjustments).
- [ ] **Number Methods:** Execute safe numeric operations using methods like parsing, rounding, formatting, and safety checking.
- [ ] **Modern Syntactic Sugar:** Use the Spread and Rest operators (`...`) for concise array destructuring, cloning, and handling variable function arguments.
- [ ] **Advanced Collections:** Differentiate and apply memory-efficient collections using `Set` and `Map`.

## 🧠 Phase 4: Execution Context & Error Resiliency

- [ ] **Hoisting:** Understand how the engine compiles and executes variable and function declarations in memory.
- [ ] **Closures:** Build an intuition for lexical scoping and state preservation inside nested function scopes.
- [ ] **Exception Handling:** Protect application runtime stability by implementing robust `try`, `catch`, `finally`, and custom `throw` logic blocks.
