## 📝 Examiner Remarks

**Score: 7/10**

**What you got right:**

- Advantages: Fast ✅, Rich library support ✅, "Foundational language" is acceptable ✅
- Disadvantages: Manual memory management ✅, No OOP support ✅

**What was missing:**

- You **did not answer the first part of the question** — "What type of language is C?"
  - C is a **middle-level, procedure-oriented (procedural), compiled, general-purpose** programming language.
  - This would have been the first line of your answer in an exam.
- You listed **3 disadvantages** but the question only asked for 2. That's fine, but the third one (no OOP) is more of a design choice than a disadvantage.
- Minor spelling errors: "Advatanges", "mangement", "independant" — these cost marks in written exams.

**Key phrase for the exam:**
> "C is a middle-level, procedural, compiled, general-purpose programming language."
